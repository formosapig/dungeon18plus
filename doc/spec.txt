怒氣 : 挨打時會增加, 能使用怒氣的職業是 Berserker, Monk
    Berserker : 每一回戰鬥增加的怒氣直接增加下一回的攻擊力.
                擁有一個特殊裝備品時能夠控制怒氣, 能控制什麼時候才將怒氣打出去.
    Monk : 每一回戰鬥增加的怒氣會存起來, 使用招式時會當成能量消耗.
----------------------------------------------------------------------------------------------------
靈魂 : 打死 npc / 英雄 / 特殊怪物等等時會收割靈魂, Assassin, Skeleton King.
    靈魂的能力能夠極力的強化能力, 並使用強力的攻擊 (黑暗波動)
    Assassin : 使用靈魂的能力是禁忌, 會墮化.
    Skeleton King : 已經墮落了, 所以使用靈魂的能力就無所謂了...
----------------------------------------------------------------------------------------------------
潛行 : 使用這個技能後, 角色的 Icon 會變半透明. Thief , Assassin.
    Thief : 必須要先潛行才能偷竊. 成長技能, 增加視野.
    Assassin : 必須要先潛行才能暗殺. 使用暗殺殺死 NPC 會獲得靈魂能量....
----------------------------------------------------------------------------------------------------
盾擊 : 架好盾牌用力往前衝, 利用盾的強韌度及衝力造成強大的傷害或破門.
    - 破門需要消耗生命力.
    - 也可以先制攻擊敵人, 消耗生命力.
    - 盾擊的強度 = 防禦力 x 盾牌指數.
    - 盾擊指數 : 黑盾[2], 白盾[0.5], 秘銀盾[0.8], 重盾[1.2] 等等...
----------------------------------------------------------------------------------------------------
門類 : 有兩種特別的數值
    - 複雜度 : 消耗更多的鑰匙.
    - 堅固度 : 需要更強的盾才能撞破.
----------------------------------------------------------------------------------------------------
視野(vision) : 決定可以發現多少可以戰鬥的事件.
----------------------------------------------------------------------------------------------------
殺價(Bargain) : 常駐成長技能, 一個固定的折扣百分比, 5% ~ 50% 。
        商人有抵抗殺的能力, 但折扣數不會變成負的.
----------------------------------------------------------------------------------------------------
暗殺 : 必須要先潛行才能使用暗殺, 以攻擊力(x2)的情況下, 能夠一擊殺死敵人視為暗殺成功.
    - 裝備暗殺者小刀時, 攻擊力(x3)
    - 裝備惡魔短刀時, 攻擊力(x4), 吸收靈魂能量
    - 裝備骷髏劍時, 攻擊力(x2.5), 吸收靈魂能量(x1.2)
    - 骷髏王、惡魔、貓妖精無法被暗殺(偵測潛行), 哭牆的暗殺攻擊力(x1)
    - 骷髏王有極大的靈魂能量. 惡魔、哭牆沒有靈魂能量, 貓妖精只有正常的靈魂能量.
